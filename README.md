# Controle de Treinos

## Status do Projeto
Em desenvolvimento

## Tecnologias Utilizadas
- Java
- NetBeans
- MySQL
- JDBC
- Git e GitHub

## Desenvolvedor
- Fran

## Objetivo do Sistema
Sistema desktop desenvolvido para realizar o controle de treinos e exercícios físicos, permitindo o cadastro e listagem de informações armazenadas em banco de dados.

## Funcionalidades do Sistema
- Cadastro de usuários
- Cadastro de exercícios
- Cadastro de treinos
- Listagem de usuários
- Listagem de exercícios
- Listagem de treinos
- Conexão com banco de dados MySQL
